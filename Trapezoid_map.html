Note:
1. If user inputs new segments manually, please make sure there is no intersection between each other.
2. Location is to show the binary search step by step and locates the left endpoint of the segment. Update is to update the new structure of Trapezoid map. If location is finished, please click Update button.
3. Mode button can change the mode between building Trapezoid map and querying a point.
4. If user wants to query another point, user only needs to click Query button again and input a new query point.
5. The number of segments in Random is 4.
6. The number of segments in Worst case is 3. Because of the uncertain number of children, the size of binary tree may be pretty large. Due to the special structure in worst case, I compact the width of the structure. There is no influence on the height of the structure. In worst case, user cannot add any segment and query a point.
